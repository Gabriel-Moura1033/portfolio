Monitoramento de Disponibilidade

Um projeto simples desenvolvido em NodeJS utilizando mssql, axios e nodemailer.

O programa realiza requisições Http para os endereços configurados e notifica por email em caso de falha.

API Escola de Inglês

Uma Api com operações CRUD para cadastro e atualização de dados de alunos e turmas.

O sistema segue o modelo MVC conectado a um banco de dados relacional através do Sequelize e gerenciando as rotas pelo Express.

API Livraria

API REST com suporte a todas as operações CRUD.

Essa API se conecta ao MongoDb através do mongoose, também utilizei o express.
